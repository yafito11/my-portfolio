# PRD – Landing Page Portfolio GenAI Engineer

## 1. Overview

Landing page ini adalah website portfolio profesional untuk **Yafie Yulianto** sebagai **GenAI Engineer** dan **AI Enthusiast**. Website bertujuan untuk:

* Menampilkan profil profesional
* Menunjukkan project berbasis AI yang pernah dibuat
* Menampilkan skill teknis di bidang GenAI
* Menarik peluang **remote work** dan **freelance project**

Website harus memiliki **UI/UX futuristik yang mencerminkan dunia AI**, dengan elemen **SVG animation**, **AI-inspired design**, dan nuansa **tech minimalism**.
